Windows 10 / 11 (Microsoft)

Windows 10 est un système d'exploitation de la famille Windows NT développé par la société américaine Microsoft. Officiellement présenté le 30 septembre 2014, il est disponible publiquement depuis le 29 juillet 2015.

Informations_Windows.exe

Windows 11 est disponible sous forme de mise à niveau gratuite vers les appareils compatibles sous Windows 10 via Windows Update, mais aussi pour les systèmes Windows 7, 8 et 8.1. Le déploiement de la mise à niveau vers Windows 11 a débuté le 5 octobre 2021 (pour les nouveaux appareils) et se poursuivra jusqu’en 2022 (pour les appareils compatibles), selon différentes annonces de Microsoft. Windows 11 est le successeur de Windows 10, sorti en 2015.

Windows, permet de réalisé n'importe quels taches, que ce soit du traitement de texte, des jeux vidéos, ou bien de simple recherche internet. Cependant, Windows est plutot gourmand en ressources systèmes ce qui ne permet pas de le lancer sur n'importe quel ordinateur
